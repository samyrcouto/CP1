# Projeto React Native - Formulario-CP1

## Sobre
Projeto focado para CP1 de mobile application, onde foi pedido a montagem de um formulário.

Utilizando essas ferramentas:

* uso de useState
* uso de useEffect
* manipulação de inputs
* organização básica de código

---

## Como funciona

O usuário preenche os campos e clica em "Enviar dados".

O app verifica se todos os campos estão preenchidos.
Se estiver tudo certo, ele mostra os dados usando um componente separado (`RenderizarDados`).
Se não, aparece um alerta pedindo pra preencher tudo.

---

## Estrutura

```
components/
  RenderizarDados.js

App.js
```
## Como rodar

1. Instalar dependências:

```
npm install
```

2. Rodar o projeto:

```
npx expo start
```

3. Abrir no celular (Expo Go) ou emulador

## Observação
O intuito do projeto foi utilizar as ferramentas aprendidas até o momento presente, melhoras futuras poderam existir no processo de aprendizagem futura.
